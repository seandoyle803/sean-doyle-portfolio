# Sean Doyle — Personal Portfolio

A one-page professional portfolio site for Sean Doyle, a Business Analytics and Information Systems student at the University of Iowa. Built as part of BAIS:3300 Digital Product Management.

## Purpose

This site replaces a traditional resume by showcasing tangible analytics projects, technical skills, and professional background in a format recruiters and hiring managers can immediately assess.

## Target Audience

Recruiters and hiring managers hiring for Data Analyst Intern and Business Analyst Intern roles.

## Tech Stack

- HTML5 (semantic elements throughout)
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (scroll-aware navigation only)
- No frameworks, no build tools, no dependencies

## Sections

- **Hero** — Chicago skyline background, name, title, headshot, CTA
- **About** — Bio with Kinnick Stadium photo
- **Skills** — Four category cards: Core Tools, Analytics, Data Visualization, Business & Supporting
- **Projects** — Unicorn Database Project, MLB Predictive Analytics Sheet, College Rating Analysis Project
- **Contact** — Email, LinkedIn, GitHub

## Running Locally

```bash
python3 -m http.server 5000 --bind 0.0.0.0
```

Then open `http://localhost:5000` in your browser.
