# Sean Doyle Portfolio

## Overview
A simple static HTML personal portfolio website showcasing projects and interests.

## Project Structure
- `index.html` - Main portfolio page
- `README.md` - Project description

## Setup & Running
- **Runtime**: Python's built-in HTTP server
- **Command**: `python3 -m http.server 5000 --bind 0.0.0.0`
- **Port**: 5000

## Architecture
Pure static HTML — no build system, no frameworks, no backend. Served directly via Python's built-in HTTP server.
